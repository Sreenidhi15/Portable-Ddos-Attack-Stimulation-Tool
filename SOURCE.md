# Firmware Source

This project uses the ESP8266 Deauther firmware by SpacehuhnTech as the base.

## Original Source
- GitHub: https://github.com/SpacehuhnTech/esp8266_deauther

## Installation Instructions

1. Download the firmware from the SpacehuhnTech repository
2. Flash it to your ESP8266 using Arduino IDE or ESP8266 Flasher
3. Follow the configuration steps in the main README
